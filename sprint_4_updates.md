## 7. Version Control & GitHub Integration
* **Repository:** [https://github.com/umair477/django_e-commerce_website.git]
* **Branch:** `master`
* **Git Workflow:**
    1. **Stage Changes:** `git add .`
    2. **Commit:** Use descriptive conventional commits (e.g., `feat: added Easypaisa payment logic`, `fix: SMTP configuration`).
    3. **Push:** Execute `git push origin master`.
* **Credentials:** Use the Environment Variables `GITHUB_TOKEN` or configured SSH keys. **Do not hardcode credentials in the source code.**
